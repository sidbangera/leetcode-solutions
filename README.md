 
## Topics Covered
 
| Topic | Description |
|---|---|
| `arrays` | Array traversal, prefix sums, in-place operations |
| `binary-search` | Search on sorted arrays / search spaces |
| `dynamic_programming` | Memoization, tabulation, classic DP problems |
| `greedy` | Locally optimal choice problems |
| `hashing` | HashMap/HashSet based lookups and frequency counting |
| `linked-list` | Singly/doubly linked list manipulation |
| `sliding-window` | Fixed and variable size window problems |
| `stack` | Stack-based problems, including monotonic stacks |
| `strings` | String manipulation and pattern problems |
| `trees` | Binary trees, BSTs, traversals |
| `two-pointers` | Two-pointer technique on arrays/strings |
 
## Naming Convention
 
Files are named as `<problem-number>-<problem-title-in-kebab-case>.java`, e.g. `704-binary-search.java`, `21-merge-two-sorted-lists.java`.
 
## Notes
 
- Solutions are written in Java.
- Focus is on understanding the underlying pattern/approach for a topic rather than just getting a problem to pass — comments are added where the reasoning isn't obvious from the code alone.
- This repo is a work in progress and updated regularly while prepping for placements.
